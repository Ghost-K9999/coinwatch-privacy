코인워치 개인정보처리방침

최종 수정일: 2026년 6월 28일

1. 수집하는 정보

코인워치는 다음 정보를 처리합니다.

  • 관심 코인/종목 목록: 사용자가 직접 추가한 코인 티커, 종목코드
  • 평단가, 알림 설정: 사용자가 직접 입력한 값
  • 결제 관련 이메일: 구독 결제 시 ExtensionPay(extensionpay.com)를 통해 수집되며, 구독 상태 확인 목적으로만 사용됩니다.

위 정보는 모두 사용자의 브라우저(chrome.storage.local)에만 저장되며, 코인워치 개발자나 제3자에게 전송되거나 판매되지 않습니다.

2. 외부 서비스와의 통신

코인워치는 다음 외부 서비스와 통신합니다. 각 서비스의 개인정보처리방침은 해당 서비스의 정책을 따릅니다.

  • 업비트(Upbit) API: 코인 시세 조회. 별도 인증 없이 공개 데이터만 요청합니다.
  • 네이버 금융: 국내 주식 시세 조회. 별도 인증 없이 공개된 페이지 정보만 읽습니다.
  • ExtensionPay / Stripe: 구독 결제 처리. 신용카드 정보는 Stripe가 직접 처리하며, 코인워치나 ExtensionPay 서버에 저장되지 않습니다.

3. 정보의 보관 및 삭제

모든 사용자 데이터는 브라우저의 로컬 저장소에만 보관되며, 확장 프로그램을 삭제하면 함께 삭제됩니다. 별도의 서버에 데이터를 보관하지 않습니다.
